# Mini Projects - Data Engineering concepts

A collection of mini projects built while learning SQL, database design, and API development in Python.

---

## Projects

### 🗓️ API and DB Integration
A beauty salon scheduling system. Creates a database to store appointment schedules and exposes an API that updates the database when a new booking is made.

**Concepts:** Database integration, REST API, CRUD operations

---

### 📚 API Design and Implementation
A library management API that allows users to search for books stored in a database.

**Concepts:** API design, database queries, search endpoints

---

### 🎮 API Requests
A Python script that fetches data from a public Pokémon API and uses it to play a card game (Top Trumps style). Two random Pokémon cards are drawn and the strongest one wins.

**Concepts:** HTTP requests, API consumption, data parsing

---

### ✈️ DB Design and Implementation
Models an airline company database from scratch and runs queries to explore and analyze the data.

**Concepts:** Database design, schema modeling, SQL queries

---

## ⚙️ Tech Stack
- Python
- Flask
- MySQL
- Requests library
